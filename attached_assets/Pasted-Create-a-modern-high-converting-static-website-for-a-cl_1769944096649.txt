Create a modern, high-converting static website for a clinical psychologist/therapist using React and Vite. This should be a single-page application (SPA) with smooth scrolling navigation between sections. NO backend or database needed - all content should be easily editable in a central config file.

TECHNICAL REQUIREMENTS:
- React 18+ with Vite as build tool
- Fully responsive (mobile-first design)
- Tailwind CSS for styling
- React Router for navigation between pages
- React Icons for iconography
- Framer Motion for smooth animations
- Ready to deploy on Vercel (include vercel.json config)
- All images should use placeholder URLs (unsplash or placeholder.com)
- Fast loading (<3 seconds)
- SEO optimized with React Helmet

DESIGN SYSTEM:
- Color Palette: Calming mental health theme
  - Primary: Soft teal/blue (#4A90A4 or similar)
  - Secondary: Warm sage green (#8B9D83)
  - Neutral: Cream/beige backgrounds (#F9F7F4)
  - Text: Charcoal gray (#2D3748)
  - Accent: Gentle coral for CTAs (#E07A5F)
- Typography: 
  - Headings: 'Poppins' or 'Inter' (clean, professional)
  - Body: 'Open Sans' or 'Lato' (readable, warm)
- Border radius: Soft rounded corners (8-12px)
- Shadows: Subtle, soft shadows for cards
- Spacing: Generous whitespace for calmness
